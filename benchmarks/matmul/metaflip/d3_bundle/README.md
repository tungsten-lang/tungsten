# 6x6 C3 x Z2 quotient lane

This directory contains the bounded, dimension-specialized Tungsten/Metal
worker for the 6x6 reversal-symmetric experiment. Its exact group action is:

```
rho(u,v,w)   = (v, transpose(w), transpose(u))
sigma(u,v,w) = (reverse(u), reverse(v), reverse(w))
```

Every mutation toggles each **distinct** image in the set orbit generated by
`rho` and `sigma`. The stabilizer check is important for fixed cubes and for
three-term C3 orbits that are already reversal-invariant. The host rejects a
seed or result unless it is factor-valid, duplicate-free, C3 closed, reversal
closed, and reconstructs the complete 6x6 matrix-multiplication tensor.

The worker ABI is `seed output walkers steps dispatches band plus_period`.
All work parameters are bounded in both `flipfleet_d3_bundle.w` and the worker.
The ordinary fleet never invokes the offline Python generator; it compiles the
checked-in `.w` and loads the checked-in `.metal` sidecar.

This engine is an experimental pool option, not a default continuous role.
The equal-compute study in `FLIPFLEET_D3_6X6.md` explains that decision.
