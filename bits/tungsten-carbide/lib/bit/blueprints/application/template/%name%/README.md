A Tungsten Carbide application.

## Getting Started

    bit install
    bit carbide db:setup
    bit carbide server

## Directory Structure

    config/         — Application configuration and routes
    lib/controllers — HTTP controllers
    lib/models      — Data models
    lib/views       — View templates
    db/migrate      — Database migrations
    spec/           — Specs
    public/         — Static assets

## Running Specs

    bit spec
